Enrich Add-on

Add-on for Visual Compositing in Blender

Enrich is the best Visual compositing add-on created for Blender. With Color Filters and Effects, you can modify any render the way you want. Every change is right before your eyes with the Enrich Panel.

Kaleidoscope Add-on

A small collection of nodes for Blender

Kaleidoscope adds few more nodes that can help you create more colorful and accurate materials. The Spectrum node allows you to generate color palettes inside Blender, and Intensity is a collection of pre defined values
