In this video I will be showing you how to Install the Kaleidoscope add-on inside Blender. Kaleidoscope add-on can be downloaded from [my site](http://blskl.cf/kaleidoscope) or on [GitHub](https://github.com/blenderskool/kaleidoscope). After downloading the .zip file, follow the steps below.

## Steps to install the add-on

- Go to File > User Preferences > Addons tab > Install from File button.
- In the File Browser, select the .zip file you downloaded. **Make sure it is in .zip file**
- Then click `Install from file` button, and choose the zip file you downloaded.
- The Kaleidoscope add-on should now be installed.

<callout>
	To see the add-on, go to **Node Editor > Materials > Shift-A (Add Nodes Menu) > Kaleidoscope**
</callout>
