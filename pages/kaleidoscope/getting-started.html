Getting Started

In this video I will be showing you how to Install the Kaleidoscope add-on inside Blender.Kaleidoscope add-on can be downloaded from my site or on GitHub. After downloading the .zip file

To install the add-on:

  • Go to File > User Preferences > Addons tab > Install from File button.
  • In the File Browser, select the .zip file you downloaded. Make sure it is in .zip file.
  • Then click Install From File button.
  • The Kaleidoscope add-on should now be installed.
  • To see the add-on, go to Node Editor > Materials > Shift-A (Add Nodes Menu) > Kaleidoscope.
