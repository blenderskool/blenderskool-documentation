1.1 Update Features

The 1.1 update of the Enrich add-on ships with a range of features and options to improve your workflow and visual experience while compositing in Blender. These new features make the Enrich add-on an essential tool, to make the most out of Blender's compositor. Now you can edit external images, image sequences directly through Enrich. The Enrich Panel in upgraded to categorize, search and favorite your presets. The new Enrich Panel is much cleaner with all the options you would need.

Changelog of the 1.1 update:

    Additions:
  • New Preset type - "Enrich Values" Allows you to save Enrich Panel settings which can be loaded later on.
  • 5 new Default presets - Bloom, Boost, Creek, Limits, and Pinch.
  • Option to use Render as Thumbnails in Presets menu.
  • Option to control the resolution of Thumbnails Generated.
  • Option to revert to Original thumbnails if render thumbnails are used.
  • Window to choose Custom Thumbnail or let Enrich Generate thumbnail for a custom Preset.
  • Option to Generate Thumbnails for a Custom Preset during its installation.
  • New Notification - "Version Fix" analyses old blend files and updates it to work with Enrich 1.1.
  • Animation support - Nearly all the Enrich Panel values can be animated.
  • Blend Mode option for Tint Settings.
  • Option to Favorite a Preset.
  • Support for Dynamic Categories (Uses Folders in the 'icon' folder for category).
  • Categories Menu to the Interface.
  • Favorites Category to the Categories menu.
  • Search Option for the Presets Menu.
  • Search Button in the Enrich Panel.
  • A Button in Preferences to open the Documentation.
  • Icon to Color Correction Settings Label.
  • Suggestions for Naming a Custom preset.
  • Suggestions for Order of Inputs and Outputs of Custom Preset.
  • Up and Down selection Buttons in Presets List.
  • Option to Refresh Presets list in User Preferences.
  • Option to Select External Image, Image sequence, Movie or Render Layer to Edit with Enrich.
  • Option to properly scale an external image while editing.
  • Scene Settings Box in the Enrich panel.
  • Option to View the Vignette Mask in Vignette Settings.
  • Tint Z-ramp to control the Z-depth map separately for Tint.
  • Option to view the Tint Z-depth Map.
  • Blur Slider in Defocus Settings.
  • Single Temperature Slider in the Color Correction Settings.
  • Option to reset the Enrich Panel to its default values.
  • Better Messages when a process is completed in the add-on.
  • Icon to be included in Custom Thumbnail generation.
    Improvements:
  • Spacing Heights in the Enrich Panel.
  • Removing Custom Preset, also removes related blend file too.
  • The Preset Export Option, which now opens the File Browser automatically.
  • Preferences Interface.
  • Notifications to work with new categories system.
  • Presets List to work with new categories system.
  • Custom Preset Installation Process.
  • Framework to load an Enrich Preset.
  • Next and Previous Preset buttons to work with new Categories system.
  • Default Thumbnails to much better thumbnails which give a better preview of Preset.
  • Enrich Interface - Boxes now render under the column, hence UI looks much cleaner.
  • Presets List - size is much bigger.
  • Size of Thumbnails in Preset thumbnails.
  • Resolution of Default thumbnails - Much faster to load.
  • The background ui_mode.txt file and other text files to a single 'settings.json' file.
  • All the add-on external web page links to blskl.cf service.
  • Save Image Interface.
  • Default values for the z-depth ramps.
  • Links Switching while viewing a z-depth ramp.
  • Grammatical errors in some parts of the Interface.
  • Z-depth ramps Interface.
  • Z-ramps are now lesser sensitive.
  • RGB Curves Interface.
  • Split Factor Default value
  • Temperature slider to a single slider. Positive is for warmth, Negative for coolness.
  • Support with other Color Managements.
  • Interface for Tint, Mist, and Defocus - Bringing all essential values to the top.
  • Load time for checking add-on updates and changelogs.
  • Default Presets code file - Runs faster.
  • Preset Export Process includes Enrich values too.
  • Removing Enrich now resets the Enrich Properties to their default values.
    Fixes:
  • Showed 'Preset Installed' after one preset is installed.
  • Error when Enrich was Generated in Edit Mode of Object.
  • Changed Icon for Lens Effects Settings.
  • The Custom Preset Installing Name Checking System.
  • Different name used when a custom Preset was installed.
  • Reloading the add-on after Custom Preset was installed.
    Removed:
  • FilePath input option for exporting presets. (File Browser opens when Export presets is clicked)
  • Temperature Settings Box (moved to Color Correction settings)
  • Sliders for Warmth, Coolness, and Strength (Merged in a single slider)
  • Defocus Node interface in Defocus Settings (Shows only Blur Slider)

I hope that you find this new update useful in your scenes in Blender. Do continue to share your thoughts about it, and new features for the future updates.
